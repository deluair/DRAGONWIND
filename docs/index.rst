======================================
DRAGONWIND Simulation Platform
======================================

.. image:: _static/images/dragonwind_logo.png
   :alt: DRAGONWIND Logo
   :align: center
   :width: 400

Welcome to the documentation for **DRAGONWIND**, a comprehensive simulation platform for China's renewable energy transition.

DRAGONWIND provides robust modeling capabilities for analyzing renewable energy expansion, grid integration, financial requirements, carbon emissions, and provincial-level impacts across China, with a focus on achieving net-zero emissions.

Features
========

- **Comprehensive Modeling**: Simulate renewable energy expansion, grid integration, financial requirements, carbon pathways, and more
- **Scenario Management**: Create and compare different policy and technology scenarios
- **Monte Carlo Simulation**: Analyze uncertainty through randomized parameter sampling
- **Interactive Visualizations**: Generate professional-quality plots and dashboards
- **Data Export**: Export results to Excel, CSV, and JSON formats for further analysis
- **Web Dashboard**: Simple UI for non-technical users to run simulations and view results
